# Course lifecycle state

Courses use a simple Draft/Published/Archived lifecycle: Draft Courses are Admin-only, Published Courses are discoverable by Learners, and Archived Courses are hidden from new Learners but remain available to already-enrolled Learners. This keeps unfinished Admin work out of the learner catalog and avoids destroying learner progress without introducing review workflows, scheduled releases, or content versioning before the app needs them.
